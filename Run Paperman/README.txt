Cara Menjalankan Paperman
1. Unzip File 'Dijalankan Pertama Kali.rar' di folder yang mudah di akses di terminal
2. Buat database di phpmyadmin dg nama paperman
3. Buka terminal/commandprompt, masuk folder itu
4. Jalankan java -jar paperman.jar
5. Setelah masuk, isi data pemilik software ini
6. Lalu masuk ke menu file > Konfigurasi > konfigurasi nomor kode buat satu kode dengan format
   'SETORAN' 'BIRU' '0'
7. Lalu masuk ke Konfigurasi > konfigurasi user, buat satu user dengan hak akses SUPER ADMIN
8. Tutup program dan jalankan file 'Dijalankan Seterusnya.rar'
9. Login dengan user name yang telah dibuat tadi
